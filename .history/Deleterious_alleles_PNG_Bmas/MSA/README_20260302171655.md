# Use VCF file to output MSA individual genes alignments.

## 1. Script 01 preprocessing

We index the VCF reference file, as the ROH pipeline does not do it.

We are going to use only SNPs to produce the alignments, indels are not use for the phylogenetic reconstruction we are doing. We will keep single-base substitutions SNPs as it does not induce change in sequence length and ambiguity about alignment. 

We use the Normalization command of bcftools
